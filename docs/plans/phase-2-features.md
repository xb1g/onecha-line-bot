# Phase 2 Features (Future)

## Overview

Features deferred from Phase 1, planned for future development.

## Features

### 1. Real-time Sync with Main Onecha App

**Status:** Deferred to Phase 2
**Reason:** Requires WebSocket/polling infrastructure

**Requirements:**
- Sync orders between LINE bot and main app
- Real-time inventory updates
- Shared customer database

### 2. Customer LINE User ID Linking

**Status:** Deferred to Phase 2
**Reason:** Requires opt-in flow, privacy policy updates

**Requirements:**
- Customer consent for ID linking
- Privacy policy compliance
- Account merging logic

### 3. Rich Analytics Dashboard

**Status:** Deferred to Phase 3
**Reason:** Beyond basic metrics, needs UI

**Requirements:**
- Visual FSM state distribution
- Conversion funnel analysis
- Revenue tracking

### 4. Multi-language Support

**Status:** Deferred to Phase 2
**Reason:** Thai-only for MVP

**Languages:**
- English
- Burmese

### 5. Additional Features (Out of Scope)

- Automated refund handling
- Voice message processing
- Group member enumeration
- Quote template customization
- Automated carrier API integration
- Bulk quote generation
